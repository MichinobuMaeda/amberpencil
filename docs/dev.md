# Development

[README](../README.md)

## Prerequisites

- git
- flutter >= 2.8.1
- node 16
- curl

## Setup

```
$ git clone git@github.com:MichinobuMaeda/amberpencil.git
$ cd amberpencil
$ yarn install
$ yarn --cwd functions install
$ yarn --cwd functions lint
$ yarn --cwd functions test
$ flutter pub get
$ flutter test
$ yarn start
```
